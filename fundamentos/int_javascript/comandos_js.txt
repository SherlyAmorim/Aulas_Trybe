FOR

Estrutura de Repetição

Sitaxe:

let variavel = valor;

for (let contador = valor inicial; contador <= valor final; contador + valor a ser acrescentado) //onde contador é uma variavel declarada para fazer o laço de repetição e o valor a ser acrescentado caso seja 1 pode ser substituído pelo ++

ou seja:

for (let contador = valor inicial; contador <= valor final; contador++) {
	console.log(variavel * contador);
}
